# BreatheLens

[English](../../README.md) | [中文](README.zh.md) | [Deutsch](README.de.md) | [Français](README.fr.md) | Русский | [Español](README.es.md) | [Português](README.pt.md) | [日本語](README.ja.md) | [한국어](README.ko.md) | [العربية](README.ar.md)

BreatheLens — локальный анализатор SD-карт ResMed CPAP / APAP. Он читает исходную папку SD-карты устройства ResMed, разбирает EDF-файлы и создает ежедневные таблицы, графики трендов, обзор утечек, Excel-отчеты и рекомендации по настройке давления.

Проект нужен тем, кто хочет быстро посмотреть данные ResMed без загрузки в облако и без импорта базы OSCAR. Выберите папку с `STR.edf`, `DATALOG` и `SETTINGS`, затем выполните локальный анализ.

## Возможности

- Выбор папки SD-карты ResMed.
- Определение модели устройства и серийного номера.
- Разбор ежедневной сводки `STR.edf`.
- Разбор длительности сеансов `DATALOG/*_PLD.edf`.
- Разбор событий `DATALOG/*_EVE.edf`.
- Показ ежедневного использования, AHI, CAI, OAI, 95 % утечки и 95 % давления.
- Переключение языка интерфейса: китайский, английский, немецкий, французский, русский, испанский, португальский, японский, корейский и арабский.
- Фоновая обработка больших папок с отображением прогресса.
- Графики AHI, 95 % утечки и 95 % давления.
- Таблицы STR, DATALOG и обзора утечек.
- Рекомендации по утечкам, достижению верхнего предела давления и центральным событиям.
- Экспорт Excel с листами `Summary`, `STR_Daily`, `DATALOG_Daily`, `Leak_Watch`, `Suggestions` и `Codebook`.
- Настольный интерфейс PySide6 + QML.
- Сборка одного исполняемого файла через Nuitka.

## Данные

| Файл | Назначение |
| --- | --- |
| `Identification.tgt` | Модель и серийный номер |
| `STR.edf` | Ежедневная сводка: AHI, утечка, давление и настройки |
| `DATALOG/*_PLD.edf` | Длительность сеансов и низкочастотные сигналы терапии |
| `DATALOG/*_EVE.edf` | Аннотации дыхательных событий |

Рекомендации основаны на трендах данных. Они не являются медицинским диагнозом. При устойчиво высоком CAI, ночной десатурации, стеснении в груди, сердцебиении или выраженной дневной сонливости обратитесь к врачу с исходными данными.

## Разработка

```powershell
uv venv .venv
uv sync
uv run python main.py
```

## Локальная сборка

```powershell
uv run python build.py
```

Результат находится в `dist/`, например `dist/BreatheLens.exe`.

## Релиз

Workflow `.github/workflows/release.yml` собирает и публикует теги `v*`. При ручном запуске можно указать `release_tag`. Для Linux x86 и Linux LoongArch64 нужны соответствующие self-hosted runners.
